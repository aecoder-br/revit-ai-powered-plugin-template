# Skill Handoff Notes

Use this reference to tailor a handoff for a concrete skill.

## Include

- Task summary.
- Role and branch.
- Owned paths.
- Changed files.
- Validation commands and results.
- Revit API context review.
- Security and data handling review.
- Remaining risks.
- Next owner and next steps.

## Avoid

- Vague validation claims.
- Unlisted generated files.
- Hidden assumptions about installed Revit versions.
- Broad ownership claims such as `src/` when only one project changed.
- Long duplicated policy text from `AGENTS.md`.
