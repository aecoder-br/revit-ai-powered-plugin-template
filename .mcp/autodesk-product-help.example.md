# Autodesk Product Help MCP

Autodesk provides a hosted Product Help MCP endpoint:

```txt
https://developer.api.autodesk.com/knowledge/public/v1/mcp
```

Use it in MCP-capable clients to search live Autodesk Help content. Client configuration schemas differ, so adapt `.mcp/cursor.mcp.example.json` to your IDE/client.

Recommended prompts:

- "List available products and supported Revit releases."
- "Search help content for Revit 2027 ExternalEvent in English."
- "Search help content for Revit 2027 MCP Server in Portuguese Brazil."
